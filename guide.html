Tạp chí (Journals)

Lấy danh sách tạp chí hoặc lọc theo từ khóa.
Tham sốKiểuMặc địnhMô tả

limitnumber50Số bản ghi trả về.
skipnumber0Bỏ qua N bản ghi đầu (phân trang).

Ví dụ

Tìm kiếm tổng hợp

GET /api/search/all hoặc POST /api/search/all

Tham sốKiểuMặc địnhMô tả
q bắt buộcstring—Câu truy vấn.
limitnumber5Giới hạn mỗi danh sách.

Ví dụ

# POST

  -H "Content-Type: application/json" \
  -d '{"q":"Education"}'

Agent API

POST /api/agent
Agent sẽ truy hồi dữ liệu (vector + text fallback) rồi tạo câu trả lời ngắn gọn bằng LLM.
TrườngKiểuMặc địnhMô tả
question bắt buộcstring—Câu hỏi người dùng.
providerstringgeminigemini hoặc qwen.
topknumber5Số tài liệu truy hồi tối đa mỗi loại.

Ví dụ

curl -s -X POST "https://neu-scholar-backend.vercel.app/api/agent" \
  -H "Content-Type: application/json" \
  -d '{
    "question":"Các hội thảo Robotics top đầu năm nay?",
    "provider":"gemini",
    "topk":5
  }'

Response mẫu

{
  "provider": "gemini",
  "answer": "… câu trả lời rút gọn …",
  "retrieved": {
    "conference": [ /* danh sách hội thảo */ ],
    "journal": [ /* danh sách tạp chí */ ]
  }
}
Ghi chú: câu trả lời phụ thuộc model & dữ liệu truy hồi.
